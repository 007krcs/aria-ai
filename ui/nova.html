What makes NOVA different from standard ARIA

Standard ARIA (CoT):
One reasoning path generated.
Final answer scored once.
Math is guessed.
No memory of prior answers.
No exploration of alternatives.
NOVA:
Multiple paths explored (MCTS).
Every step scored (PRM).
Math executed in Python.
Checks against belief store.
Best path selected, not first path.
